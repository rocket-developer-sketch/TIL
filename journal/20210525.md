3. NULL
```
- 연산
    - +, -, *, / 등 : NULL
    - >, >=, =, <= ,< 등 : FALSE
    - AVG, SUM 등 : NULL인 행 제외하고 행 개수 COUNT, NULL 인 행 제외하고 데이터 연산

- NULL 값 찾기
    - IS NULL
    - IS NOT NULL
    
- IN(NULL) => NULL 데이터는 출력되지 않음

```
